# Sports Core

Dark premium sports website built with Next.js 14 App Router, Tailwind CSS, Framer Motion, and shadcn-style UI components.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Routes

- `/` Home
- `/f1` F1 race calendar
- `/f1/race/[round]` F1 race detail
- `/football` Football fixtures + standings
